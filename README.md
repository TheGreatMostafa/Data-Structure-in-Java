# Java Data Structures

Welcome to my personal collection of data structures, implemented from scratch in Java. This repository is a journey through the foundational concepts of computer science, built with a focus on creating clean, efficient, and well-documented code.

---

## 🎯 Goals

The primary objective of this repository is to deepen my understanding of data structures by building them from the ground up. The goals are:

* **To Master the Fundamentals:** Solidify knowledge of how core data structures work under the hood.
* **To Build a Strong Portfolio:** Showcase practical coding skills and an understanding of key CS concepts.
* **To Practice Clean Code:** Write readable, maintainable, and efficient Java code following best practices.
* **To Track Progress:** Document my learning journey on a weekly basis.

---

## 📂 Repository Structure
Each data structure has its own folder with:
- Java implementation file(s).
- Example usage in a main() method.
- A short README

---

## 📈 Current Progress

I plan to add a new data structure implementation each week. Here is the current progress:

* [x] **List:** A dynamic array-based list (`MyList`) that supports generic types.
* [ ] **Stack:** LIFO (Last-In, First-Out) stack implementation.
* [ ] **Queue:** FIFO (First-In, First-Out) queue implementation.
* [ ] **Linked List:** Singly and Doubly Linked List implementations.
* [ ] **Binary Search Tree:** A node-based binary tree.
* [ ] **Hash Table:** A hash map with collision handling.

## 🚀 How to Run the Code

You can clone this repository and run the code on your local machine.

## 🚀 How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/itsmostafa07/Data-Structure-in-Java
   ```

2. Navigate to the desired folder and compile:

   ```bash
   javac MyList.java
   java MyList
   ```
